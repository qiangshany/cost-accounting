# 氯碱厂成本管理系统 - 完整需求文档

## 一、项目概述

### 1.1 项目背景
某氯碱厂需要进行生产成本管理与分析，系统需要支持多个部门的成本数据填报、汇总计算、Excel导出等功能。

### 1.2 系统目标
- 实现生产成本的精细化管理
- 支持多部门协同填报（车间、采购部、生产管理部）
- 提供成本分析与可视化展示
- 支持数据导出与报表生成

### 1.3 用户角色
| 角色 | 职责 | 权限范围 |
|------|------|----------|
| 碱车间 | 填报直接材料数量、制造费用 | 只能填报碱车间相关数据 |
| 采购部 | 填报原材料采购单价 | 只能填报采购单价 |
| 生产管理部 | 填报期间费用、调整项、部分人工成本 | 只能填报期间费用、调整项 |
| 管理员 | 查看所有数据汇总、成本分析、导出Excel | 全局数据查看、导出 |

### 3.2 成本计算逻辑

#### 3.2.1 原材料成本计算
```
原材料成本 = Σ(材料数量 × 对应单价)
```

#### 3.2.2 成本分配逻辑（按纯碱浓度比例）
```
总碱产量 = 32%烧碱产量 × 0.32 + 50%烧碱产量 × 0.5

32%烧碱应分配成本 = 总成本 × 0.53 × (32%烧碱产量 × 0.32 / 总碱产量)
50%烧碱应分配成本 = 总成本 × 0.53 × (50%烧碱产量 × 0.5 / 总碱产量)

32%烧碱吨成本 = 32%烧碱应分配成本 / 32%烧碱产量
50%烧碱吨成本 = 50%烧碱应分配成本 / 50%烧碱产量
```
